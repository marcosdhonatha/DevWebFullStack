Esse é um exemplo com as tags:

Tags vistas na aula:

  • strong/b para textos em negrito
  • em/i para textos em itálico
  • mark para textos com marcação padrão
  • small para texto pequeno
  • del para texto "apagado"
  • ins para texto sublinhado/inserido
  • sub para texto subscript
  • sup para texto supscript
