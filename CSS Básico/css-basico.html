CSS Básico.

Nesse modulo vamos aprender várias coisas básicas sobre CSS.
Preferi fazer todas as aulas em um unico arquivo.

Definir um id:

<div id="nome-ID"> - elementos- </div>

Usando o id no CSS:

#nome-ID{ estilos CSS }

O ID é único para cada elemento, para usar o mesmo CSS em vários elementos usamos classes.

Definindo uma Classe:

<div class="nome-da-classe"> - elementos - </div>

Usando a Classe no CSS:

.nome-da-classe{ - estilos css - }
