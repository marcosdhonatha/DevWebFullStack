Criando um visualizador de videos no youtube:

<form method="get" action="https://www.youtube.com/watch"> <input type="text" name="v"> <input type="submit"> </form> O segredo é colocar o nome do camopi do formulário como "v" tal qual é feito no youtube.
Além, disso o valor informado deve ser válido para algum video do youtube.

Ex: q_xeCu3P9hU

Criando um buscador de videos no youtube:

<form method="get" action="https://www.youtube.com/results"> <input type="text" name="search_query"> <input type="submit"> </form> Devemos colocar a tag action como "https://www.youtube.com/results" Tal qual é feito no youtube.br
Além disso, o campo da busca deve ter o nome de "search_query", como no youtube.

Tags semânticas

  • header: Cabeçalho da página.

  • main: Uma por página, todo o conteúdo principal deve ficar dentro dela.

  • footer: Rodapé da página.

  • section: É bom cada seção do site ter uma tag dessa.

  • article: Cada artigo, cada pedaço de texto que faria sentido mesmo fora da página deve ter.

  • nav: Navigation, "menu" de navegação.
